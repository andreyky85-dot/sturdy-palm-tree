# Assets for Content Multiplier

## og.png (превью в соцсетях)

- **Нужный размер:** 1200×630 px.
- Если картинка уже лежит в этой папке как `og.png`, приведите её к размеру одной командой:

```bash
npm run resize-og
```

Скрипт обрежет по центру до 1200×630 и перезапишет `public/og.png`. (Требуется установленный `sharp` — он есть в devDependencies.)

## Остальные файлы

2. **favicon.ico** — 32×32 или 48×48. Иконка во вкладке браузера.  
   Next.js отдаёт его по адресу `/favicon.ico`, если файл лежит здесь.

3. **apple-touch-icon.png** — 180×180 (по желанию). Для «Добавить на главный экран» на iPhone.

После добавления `og.png` превью в соцсетях будет использовать эту картинку.
